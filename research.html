2026-04-20 Critical Exposure Analysis

FortiClient EMS Internet Exposure — CVE-2026-35616 Remediation Analysis

CVE-2026-35616 — CVSS 9.8

Following Fortinet's disclosure of a critical unauthenticated RCE vulnerability in FortiClient Enterprise Management Server (affected versions 7.4.5–7.4.6, fixed in 7.4.7), we conducted passive internet exposure analysis and behavioral probe validation across 100 confirmed EMS instances. We identified healthcare organizations, municipal governments, and academic institutions operating EMS past the CISA mandatory remediation deadline of April 9, 2026. We developed a behavioral detection methodology that distinguishes patched from vulnerable hosts without authentication. All tested named organizations have since remediated.

2026-04-09 High AI/ML Security

AI Framework Vulnerabilities — Seven Findings Across Major Agentic Libraries

smolagents · MLflow · Haystack · AutoGen · LlamaIndex · OceanBase — CVEs pending

Seven vulnerabilities identified across widely-deployed AI framework libraries: a format string bypass and remote executor pickle RCE in smolagents, an authentication bypass in MLflow online scoring, an arbitrary class instantiation RCE in Haystack, a module injection vulnerability in AutoGen's load_component, a workflow deserialization vulnerability in LlamaIndex, and a SQL injection in OceanBase. All findings are in coordinated disclosure with affected vendors.

Research areas

AI/ML framework security

Vulnerability research on the libraries that power AI applications — LLM frameworks, agent runtimes, ML pipeline infrastructure. This is a fast-moving surface with significant undiscovered exposure.

Infrastructure exposure analysis

Systematic analysis of internet-exposed enterprise infrastructure following high-severity CVE disclosures. We identify who is exposed, attribute organizations where possible, and notify through appropriate channels.

AI system threat modeling

Research into the attack surface of deployed AI systems — prompt injection, indirect instruction injection, data exfiltration through model context, authorization bypass in agent workflows.

Methodology and tooling

Detection methodologies, fingerprinting techniques, and reusable tooling developed during research engagements. Published under open license where disclosure doesn't create undue risk.
